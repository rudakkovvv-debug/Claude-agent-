# Personal Brand System

Инструменты для построения личного бренда с нуля и упаковки его в готовый
контент — через Claude.

## Что внутри

- **brand-blueprint-ru** — позиционирование с нуля: Big Idea, аудитория,
  враг и отличие, личность, контент-план. Разговор целиком на русском.
- **content-engine-ru** — берёт уже готовый Бренд-Бриф и превращает его в
  готовые сценарии YouTube, Reels и посты в Telegram (не идеи — готовый
  текст).
- **brand-blueprint** — та же позиционирующая часть на английском.

## Установка (без терминала, 2 минуты)

Работает в **Cowork** — доступен на любом платном тарифе Claude, отдельно
устанавливать ничего не нужно, кроме самого плагина.

1. Открой Cowork → в боковой панели **Customize** → **Plugins**.
2. Нажми **Add marketplace**, вставь: `<владелец>/personal-brand-system`
   (ссылка на этот репозиторий).
3. В списке появятся три плагина — нажми **Install** у нужных.
4. Всё, готово. Напиши в чат «создай мой личный бренд» — начнётся разговор.

## Порядок использования

Сначала `brand-blueprint-ru` (или `brand-blueprint` для английского) — это
даёт Бренд-Бриф. Затем, если нужны готовые сценарии/посты — `content-engine-ru`
берёт этот же Бриф и производит контент. Без готового Брифа `content-engine-ru`
работать не будет — это осознанное ограничение, а не баг: контент без
устоявшегося позиционирования — это ровно тот общий контент, от которого весь
этот проект должен избавить.

## Для тех, кто использует Claude Code (CLI)

```bash
/plugin marketplace add <владелец>/personal-brand-system
/plugin install brand-blueprint-ru@personal-brand-system
/plugin install content-engine-ru@personal-brand-system
```
